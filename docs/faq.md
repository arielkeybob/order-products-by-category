# Perguntas Frequentes

Aqui estão algumas perguntas frequentes sobre o Plugin de Ordenação de Produtos por Categoria.

## A ordenação afeta a performance do site?

Não, o plugin foi otimizado para não impactar o desempenho do site.

## Posso definir a ordem dos produtos individualmente para cada categoria?

Sim, a ordenação é feita em uma base por categoria.

## O plugin suporta ordenação em massa?

Atualmente, a ordenação é feita manualmente através da interface de arrastar e soltar.

## Como posso contribuir para o plugin?

Consulte o arquivo `contributing.md` para mais informações sobre como você pode contribuir.
